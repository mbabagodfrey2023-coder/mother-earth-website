Environmental intelligence for East Africa and beyond. Version 1.0

Early Access: IRIS API is currently onboarding its first cohort of subscribers. To get your API key, email hello@motherearth.systems or subscribe at motherearth.systems.

Overview

The IRIS API gives you programmatic access to Mother Earth's environmental data feeds — air quality, carbon sequestration, water quality, soil health, biodiversity indices, and deforestation alerts across Africa and beyond.

Every data point is validated by GAIA, our environmental intelligence AI, and carries a cryptographic certificate of origin secured by NEXUS on-chain. What you receive is what was measured — unmodified.

Authentication

All API requests require your IRIS API key in the Authorization header.

# Every request must include this header
Authorization: Bearer iris_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
Never expose your API key in frontend JavaScript or public repositories. Store it server-side only.

Example — cURL

curl https://api.motherearth.systems/v1/feeds \
  -H "Authorization: Bearer iris_live_your_key_here"

Example — Node.js

const axios = require('axios');

const iris = axios.create({
  baseURL: 'https://api.motherearth.systems/v1',
  headers: { Authorization: `Bearer ${process.env.IRIS_API_KEY}` }
});

const feeds = await iris.get('/feeds');
console.log(feeds.data);

Example — Python

import requests, os

headers = {"Authorization": f"Bearer {os.environ['IRIS_API_KEY']}"}
resp = requests.get("https://api.motherearth.systems/v1/feeds", headers=headers)
print(resp.json())

Create an Alert

Sentinel +
POST/alerts

Receive a webhook or email when a threshold is crossed.

# Example: alert when PM2.5 > 35 µg/m³ in Nairobi
curl -X POST https://api.motherearth.systems/v1/alerts \
  -H "Authorization: Bearer iris_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "feed_id":    "air_quality_nairobi",
    "metric":     "pm25",
    "condition":  "gt",
    "threshold":  35,
    "notify": {
      "webhook_url": "https://your-server.com/iris-alert",
      "email":       "alerts@yourorg.com"
    }
  }'
