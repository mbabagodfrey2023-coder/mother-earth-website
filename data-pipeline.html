National Oceanic and Atmospheric Administration

The Mauna Loa Observatory in Hawaii has measured atmospheric CO₂ continuously since 1958 — the longest direct record on Earth. Instruments sample air every hour. Readings are peer-reviewed weekly and published by the NOAA Global Monitoring Laboratory. The number you see on this site comes directly from that dataset.

NASA EONET
Earth Observatory Natural Event Tracker

NASA's real-time event catalogue aggregates satellite observations from the Terra, Aqua, Suomi-NPP, and NOAA-20 satellites. It tracks wildfires, severe storms, volcanoes, sea ice loss, and dust events — globally, 24 hours a day. Each event includes GPS coordinates, category, satellite source, and timestamp.

NASA FIRMS
Fire Information for Resource Management System

MODIS and VIIRS instruments aboard NASA's polar-orbiting satellites detect thermal anomalies — active fire pixels — globally within 3 hours of a satellite overpass. FIRMS has been detecting wildfires from space since 2000. Every wildfire reading on this platform is a confirmed thermal detection, not a weather-model estimate.

NASA GISTEMP
Goddard Institute for Space Studies

The GISS Surface Temperature Analysis combines land station data from GHCN-v4 and ocean data from ERSST v5 — over 6,300 weather stations worldwide — to calculate global mean temperature relative to the 1951–1980 baseline. The +1.2°C anomaly is the latest published global mean, updated monthly by NASA scientists.

// THE PIPELINE

Six Steps from Satellite
to Your Screen

GAIA is the environmental intelligence agent that runs this pipeline. Every 15 minutes, automatically, without human intervention.

01
// TRIGGER

GAIA Agent Wakes Up

A Cloudflare Cron Trigger fires every 15 minutes — 96 times per day, every day of the year. GAIA's run() function is called. No human schedules it. No server is always running. Serverless compute wakes, runs the cycle, and sleeps.

// Cloudflare Worker scheduled trigger
export default {
  async scheduled(event, env, ctx) {
    // Fires every */15 * * * * (cron)
    await runGaiaCycle(env);
  }
}
02
// FETCH

Live API Requests to NASA & NOAA

GAIA makes parallel HTTP requests to four government APIs simultaneously. No auth tokens needed — NASA EONET and NOAA CO₂ data are publicly available. The raw responses are JSON and CSV, returned in milliseconds from government servers.

// Parallel fetch — all four sources at once
const [eonet, co2, gistemp, firms] = await Promise.all([
  fetch('https://eonet.gsfc.nasa.gov/api/v3/events?status=open'),
  fetch('https://gml.noaa.gov/webdata/ccgg/trends/co2/co2_weekly_mlo.csv'),
  fetch('https://data.giss.nasa.gov/gistemp/tabledata_v4/GLB.Ts+dSST.csv'),
  fetch('https://firms.modaps.eosdis.nasa.gov/api/area/csv/...')
]);
⚡ ALL FOUR IN PARALLEL — TYPICALLY <800ms TOTAL
03
// PARSE & VALIDATE

Data is Parsed and Sanity-Checked

Raw CSV and JSON are parsed. GAIA extracts the specific values: the latest CO₂ weekly reading (skipping comment lines), the most recent GISTEMP anomaly, the count of open EONET events by category, and FIRMS fire pixel count. Obvious outliers — a CO₂ reading of 9,000 ppm or negative wildfires — are flagged and the previous reading is kept.

// Parse CSV — skip header rows starting with '#'
const lines = csv.split('\n').filter(l => !l.startsWith('#'));
const latest = lines[lines.length - 2].split(',');
const co2_ppm = parseFloat(latest[4]); // column 4 = weekly mean

// Sanity check
if (co2_ppm < 300 || co2_ppm > 600) throw 'Outlier — keeping previous';
04
// AI SYNTHESIS

Claude AI Reads the Numbers

The parsed data is passed to Claude — the same AI that powers all 33 agents. Claude compares the new reading to the previous cycle, detects significant changes (a wildfire outbreak, a CO₂ spike, an unusual temperature anomaly), and generates a natural-language mission log entry. This is what you see in the Mission Log on the platform.

// Send to Claude for synthesis
const analysis = await claude.messages.create({
  model: 'claude-sonnet-4-5',
  messages: [{
    role: 'user',
    content: `GAIA cycle. CO₂: ${co2_ppm} ppm (prev: ${prev}). Events: ${events}. Synthesise.`
  }]
});
// Returns: "CO₂ held at 432.03 ppm. 7 active NASA events tracked globally."
🤖 CLAUDE SONNET — EVERY CYCLE
05
// STORE

Written to Cloudflare KV & live-data.json

The validated, AI-synthesised data object is written to two places simultaneously: Cloudflare Workers KV (a global key-value store with sub-millisecond reads) and a static live-data.json file served from the CDN. Both are updated atomically. The timestamp records exactly when this cycle ran.

// Write to KV — globally distributed, <1ms read latency
await env.LIVE_DATA.put('current', JSON.stringify({
  updated: new Date().toISOString(),
  environmental: { co2_ppm, temp_anomaly_c, active_nasa_events, wildfires },
  platform: { agents_online: 33, branches_active: 7 }
}));
// Also writes /live-data.json for browser direct fetch
06
// DELIVER

Cloudflare Serves It to Your Browser

When you load any page on motherearth.systems, your browser fetches /live-data.json from the nearest Cloudflare edge node — one of 300+ points of presence worldwide. The data travels from NASA's servers in Greenbelt, Maryland → GAIA's compute in Cloudflare's network → your screen. End-to-end latency: typically under 100ms from wherever you are.

// Browser-side fetch (this page does this right now)
const res = await fetch('/live-data.json');
const data = await res.json();
document.getElementById('co2-display').textContent =
  data.environmental.co2_ppm + ' ppm';

Why You Can Trust These Numbers

Every architectural choice in this pipeline was made to eliminate the possibility of manipulated or cherry-picked data.

🏛️

Government Primary Sources Only

NASA and NOAA are independent government agencies with no commercial stake in the numbers. Mauna Loa CO₂ data has been maintained by Scripps Institution of Oceanography and NOAA continuously since 1958 — before climate was politically contentious.

🔗

Direct API — No Intermediaries

GAIA fetches from the authoritative source endpoint directly. There is no third-party data broker, no aggregator, no repackaged feed between NASA's servers and this platform. The URL in our code is the same URL you can open in your browser right now.

⏱️

Timestamped and Immutable

Every data cycle writes an ISO 8601 UTC timestamp. The Mission Log on the platform is append-only — past entries cannot be edited. You can see exactly when each reading was recorded and compare it to the NOAA source to verify it yourself.

🤖

AI Synthesis, Not AI Invention

Claude synthesises the data into language — it does not generate the numbers. The CO₂ reading is not a model output, not a prediction, not an AI estimate. It is a sensor reading. Claude's role is to explain it, not to produce it.

🔒

Purpose Trust Governance

The Golden Share held by the Purpose Trust contains an explicit protected matter: data provenance and integrity cannot be altered by any board resolution, commercial arrangement, or investor instruction. No one can be paid to change a number.

👁️

Publicly Auditable

The source code for the GAIA pipeline is on GitHub. The API endpoints are published on this page. The live-data.json file is openly accessible. Any journalist, researcher, or curious visitor can independently verify that what we publish matches what NASA published.
