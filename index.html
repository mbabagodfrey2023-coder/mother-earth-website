Blockchain-Secured Provenance
NEXUS FORGE KEYS IRIS

How provenance works

1. GAIA validates data point (VERIFY — 3-layer check)
2. NEXUS FORGE hashes: data + timestamp + sensor ID
3. Hash anchored to blockchain — immutable record
4. KEYS signs with air-gapped hardware key
5. Certificate issued — any buyer can verify independently

What this solves

  • Carbon credit fraud — verified sensor data, not self-reported claims
  • Data manipulation — tamper-evident audit trail from source to sale
  • Legal standing — on-chain records admissible under Kenya Evidence Act
  • Cross-border trust — any regulator can audit without intermediary

On the IRIS marketplace

Every dNFT listed on the IRIS data marketplace carries a NEXUS provenance certificate. Buyers see the full chain: sensor → validation → anchor → sale. No black boxes.
