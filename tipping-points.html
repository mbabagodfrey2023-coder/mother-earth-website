Methodology
Baselines and critical thresholds from IPCC AR6 (2021). Stress scores (0–100) measure deviation from pre-industrial or 1981–2010 reference values. Cascade probability amplifies when multiple high-stress (≥65) tipping points are connected in the cascade network — each active link adds 3.5%. Four tipping points use live API feeds (NOAA GML, NASA GISTEMP, NSIDC). Five use proxy derivations from live data and published rates. Phase 2 (Fable 5): All proxies will be replaced with direct sensor feeds and ML inference on raw observational data.
